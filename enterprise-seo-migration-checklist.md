# Enterprise SEO Migration Checklist

A framework for managing large-scale website migrations while protecting organic traffic and search visibility.

---

## Pre-Migration Planning

- Crawl the existing site (Screaming Frog, Sitebulb, etc.)
- Export all indexable URLs
- Identify top traffic pages
- Identify high-value backlink pages
- Benchmark rankings and organic traffic
- Export current XML sitemaps
- Document canonical tag usage
- Document hreflang structure

---

## URL Mapping

- Create redirect mapping from old URLs to new URLs
- Avoid redirect chains
- Avoid redirect loops
- Validate status codes (301 vs 302)

---

## Technical Validation

- Validate canonical tags
- Validate hreflang implementation
- Update XML sitemaps
- Validate robots.txt
- Confirm internal linking structure
- Check structured data implementation

---

## Launch Monitoring

- Crawl the new site immediately after launch
- Validate redirects
- Check indexation in Google Search Console
- Monitor crawl errors
- Track ranking volatility

---

## Post-Migration Monitoring

- Monitor organic traffic trends
- Monitor keyword rankings
- Review indexing coverage
- Monitor Core Web Vitals
- Validate structured data

---

## Key Tools

- Screaming Frog
- Google Search Console
- GA4
- Adobe Analytics
- Ahrefs
- Semrush
- Log file analysis tools
