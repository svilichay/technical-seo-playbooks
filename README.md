# Technical SEO Playbooks

This repository contains frameworks, checklists, and technical implementation examples used to support scalable search performance.

Topics include enterprise website migrations, structured data implementations, crawl optimization, and emerging AI search optimization practices.

These playbooks are designed to help technical SEO teams, developers, and product teams implement search best practices at scale.

---

## Topics Covered

### Technical SEO Audits
Frameworks for identifying crawl, indexation, and technical performance issues.

### Website Migration SEO
Processes for managing domain migrations, CMS migrations, and large-scale URL restructuring while protecting organic traffic.

### Structured Data Implementation
Schema strategies for enhancing search visibility, including examples for:

- Organization schema
- Article schema
- Product schema
- FAQ schema

### AI Search & Answer Engine Optimization (AEO)
Emerging strategies for optimizing content and entities for AI-driven discovery.

### Crawl & Indexation Optimization
Best practices for managing crawl budgets, canonicalization, and indexation signals.
